# Reviewer Assignment Service  
Микросервис назначения ревьюеров для Pull Request’ов  
**Backend Internship, осенняя волна 2025**

---

## 1. Описание

Проект реализует сервис для автоматического назначения ревьюеров на Pull Request’ы, управления командами и пользователями, а также выполнения операций merge и переназначения.  
Стек: **Java, Spring Boot, PostgreSQL, Docker Compose**.

---

## 2. Несоответствие OpenAPI

Данная реализация **не соответствует** предоставленной OpenAPI-спецификации.  

- Ниже перечислены **фактические действующие эндпоинты**.
- В репозитории присутствует ветка **`api/`**, содержащая черновую адаптацию под OpenAPI.  
  Эта версия **не протестирована**, корректность работы не гарантируется.

---

## 3. Запуск проекта

### Требования
- Docker + docker-compose  
- Порт 8080 должен быть свободен

### Команда запуска
```bash
docker-compose up --build
```
## Список эндпоинтов сервиса

### Users API
| Метод | Маршрут | Описание |
|-------|---------|----------|
| POST | /users | Создать пользователя |
| GET | /users/{id} | Получить пользователя |
| GET | /users | Получить всех пользователей |
| GET | /users/by-team/{teamId} | Получить пользователей команды |
| PATCH | /users/{id}/activity | Обновить активность пользователя |

### Teams API
| Метод | Маршрут | Описание |
|-------|---------|----------|
| POST | /teams | Создать команду |
| GET | /teams | Получить все команды |
| GET | /teams/{id} | Получить команду по ID |
| PUT | /teams/{id} | Обновить команду |
| DELETE | /teams/{id} | Удалить команду |

### Pull Request API
| Метод | Маршрут | Описание |
|-------|---------|----------|
| POST | /pull-request | Создать PR |
| GET | /pull-request/{id} | Получить PR |
| POST | /pull-request/{id}/reassign?oldReviewerId=UUID | Переназначить ревьювера |
| POST | /pull-request/{id}/merge | Merge PR |
| GET | /pull-request/reviewer/{userId} | PR’ы ревьюера |

### Stats API
| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | /stats/reviewers | Статистика по ревьюерам |
| GET | /stats/pull-requests | Статистика по PR |
